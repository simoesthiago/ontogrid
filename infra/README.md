# Infra local

O bootstrap inicial de infraestrutura do OntoGrid usa apenas o arquivo [docker-compose.yml](/C:/Users/tsimoe01/coding/ontogrid/docker-compose.yml).

Serviços previstos:

- API FastAPI
- Frontend Next.js
- PostgreSQL/TimescaleDB
- Neo4j
- Redis

Este diretório existe para receber variações futuras de compose, manifests e notas operacionais sem poluir a raiz do repositório.
